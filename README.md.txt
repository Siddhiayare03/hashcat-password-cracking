# Hashcat Practice & Password Hash Analysis

This repository documents my hands-on practice with Hashcat to understand
password hashing, cracking techniques, and common error handling.

⚠️  – No real user data is used.



## 🔧 Environment
- OS: Windows 10
- Tool: Hashcat v7.1.2
- GPU: NVIDIA RTX 3050 Laptop GPU
- Wordlist: rockyou.txt
- Rules: best66.rule



##  Concepts Covered
- Hash identification
- Dictionary attacks
- Rule-based attacks
- Mask attacks
- Hashcat status meanings
- Error debugging and fixes


##  Learning Outcome
- Understood how password hashes work
- Learned why some hashes crack and some don’t
- Gained experience handling real-world Hashcat errors
- Improved command-line and cybersecurity fundamentals
